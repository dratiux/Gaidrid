A premium browser New Tab workspace. Modular, keyboard-first dashboard with 12 themes and 10+ widgets.

## Quick Start

```bash
git clone https://github.com/dratiux/Gaidrid.git
cd Gaidrid
npm install
npm run dev
```

## Install as Extension

1. `npm run build`
2. Open `chrome://extensions` or `edge://extensions`
3. Enable **Developer Mode**
4. **Load unpacked** → select `dist/`

Notes · Todo · Calendar · Bookmarks · Finance · News · Sports · Games · Weather · Pomodoro
